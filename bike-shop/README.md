# Bike Rental Shop

Solution for the [bike rental shop curse](https://www.freecodecamp.org/learn/relational-database/learn-bash-and-sql-by-building-a-bike-rental-shop/build-a-bike-rental-shop).

- `setup.sql` sets up a postgres database
- `bike-shop.sh` is an interactive bash CLI to rent/return bikes and ad customers
